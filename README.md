# 🎈 射氣球大作戰 (Balloon Shooter)

無敵星星音效版 — 20 秒內射破氣球、收集掉落物拿最高分！

## ▶️ 線上試玩
打開 GitHub Pages 網址即可遊玩 (支援 PC 滑鼠與手機觸控)。

## 🎮 玩法
- PC：滑鼠移動準心，左鍵點擊射擊
- 手機：直接點擊目標
- 每局 20 秒

## 💯 積分
| 物件 | 分數 |
| :-- | :-- |
| 🎈 氣球 | +1 |
| 🎁 禮物 | +5 |
| 💣 炸彈 | -10 (無敵模式 +5) |
| ⭐ 星星 | 觸發 5 秒無敵 |

無敵模式：分數加倍、炸彈免疫、背景變金色、有節奏背景音。

## 🛠️ 技術
- 純 HTML5 Canvas + Vanilla JS
- Web Audio API 程式合成音效 (無 mp3)
- 單一 `index.html`，開啟即玩
